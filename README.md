# Trabajo Práctico Integrador (TPI) - Programación 1 🚀
Universidad Tecnológica Nacional (UTN) - Tecnicatura Universitaria en Programación

## Descripción del Programa
Este proyecto es una aplicación de consola en Python diseñada para gestionar información sobre países. Permite leer datos desde un archivo CSV, agregar, actualizar, buscar, filtrar y ordenar países según distintos criterios (como continente, población o superficie), y generar estadísticas básicas.

## Instrucciones de Uso
1. Descargar o clonar este repositorio.
2. Asegurarse de tener instalado Python 
3. Ejecutar el archivo principal desde la terminal: 
4. El programa cargará automáticamente los datos del archivo `paises.csv`.
5. Navegar por el menú interactivo ingresando el número de la opción deseada. Al finalizar, seleccionar la opción 7 para guardar los cambios y salir.

## Ejemplos de Entradas y Salidas
Ejemplo - Buscar País:
- Entrada: Se selecciona la opción 3 y se ingresa "arg".
- Salida: --------------------
  Nombre: Argentina
  Población: 45376763
  Superficie: 2780400
  Continente: América

**Ejemplo - Estadísticas:**
- *Entrada:* Se selecciona la opción 6.
- *Salida:* Muestra el país con mayor/menor población, el promedio de superficie y el recuento de países por continente.
